STEP 1: COMPUTE FEATURES (for both A and B)

For each equation E with sides L=R, count variable letters per side. Then:
- `size(E)` = total variable occurrences
- `vars(E)` = number of distinct variables (letters only; multiplicity ignored, e.g. `x*y = (x*(x*z))*x` has vars=3)
- `imb(E)` = sum over each variable of |count_left − count_right|
- `bare(E)` = one side is a single variable
- `LP(E)` = leftmost vars match across the two sides
- `RP(E)` = rightmost vars match
- `SET(E)` = both sides use the same variable set
- `XOR(E)` = every variable has matching parity on both sides
- `AB(E)` = every variable has exactly the same count on both sides

Worked example, `x = y*(y*((x*y)*x))`: left vars `[x]` (count 1), right vars `[y,y,x,y,x]` (count 5). `size=6`, `vars=2`, `imb=|1-2|+|0-3|=4`, `bare=TRUE`, `LP=FALSE`, `RP=TRUE` (both rightmost are `x`), `SET=FALSE`, `XOR=FALSE`, `AB=FALSE`.
